```
□ Acceptance Criteria pass (theo spec.md)
□ Code reviewed & approved trên PR
□ Lint + typecheck + unit test pass
□ Security scan pass
□ Integration/E2E test pass (theo risk level)
□ API documentation cập nhật
□ Architecture doc đồng bộ nếu đổi kiến trúc
□ Rollback validated (High/Critical)
□ UAT approved bởi PO/BO
□ Git Issue đóng
```
